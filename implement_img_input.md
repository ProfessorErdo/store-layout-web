### Objective
take an image of store shelf layout and implement it into the web application

### Steps
1. implement the image import feature in the web app 
2. analyze the image using ollama qwen3-vl model to extract the store shelf layout info including the shelf position, shelf orentation, shelf length, shelf product and etc. which can be used to generate the store shelf layout
3. save the info into a json file
4. use the info to generate the layout in the web app
